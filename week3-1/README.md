# week3-1

## 주제

전통 브라우저 script 로드 방식과 모던 JavaScript 개발 환경(Node/npm/package.json)을 비교하며, 프론트엔드 실행 환경의 기본 체력을 만드는 회차다.

## 진행 목표

- 전통 script 태그 방식의 한계와 모던 프로젝트 구조의 필요성을 설명한다.
- Node.js와 npm의 역할을 구분하고 `package.json`의 핵심 필드를 읽는다.
- `npm install`, `npm run dev`, `npm run build` 흐름을 실습 기준으로 설명한다.
