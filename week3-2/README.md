# week3-2

## 주제

Vue 기초와 상태 기반 UI 사고 전환을 실제 컴포넌트 예제로 설명하는 회차다.

## 진행 목표

- Vue 단일 파일 컴포넌트의 기본 구조를 읽는다.
- `v-model`, `@click`, `v-if`, `v-for`가 상태 변화와 어떻게 연결되는지 설명한다.
- 검색/필터/상세/등록 예제를 기준으로 상태 기반 UI 흐름을 시연한다.
