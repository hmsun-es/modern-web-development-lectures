# 현대 웹 개발 강좌 for ESGROUP

이 저장소는 회차별로 바로 열어서 강의를 진행할 수 있도록 재구성한 강의 워크스페이스다.

각 회차 폴더는 다음 3가지를 공통으로 가진다.

- `README.md`: 해당 회차 진행 목적, 추천 순서, 핵심 파일 안내
- `docs/`: 강의 문서, 슬라이드 초안, 설계 참고 자료
- `src/`: 실습 코드 또는 회차별 소스 안내

## 회차별 폴더

### week1-1

- 주제: Git 기초, VS Code Source Control, 프로젝트 킥오프
- 시작 파일: `week1-1/README.md`

### week1-2

- 주제: GitHub Flow, Pull Request, 리뷰와 병합
- 시작 파일: `week1-2/README.md`

### week2-1

- 주제: ASP.NET Core Web API 시작, CLI, DI, Middleware, Swagger
- 시작 파일: `week2-1/README.md`

### week2-2

- 주제: EF Core, DbContext, Tracking, SQL 확인, 진단 흐름
- 시작 파일: `week2-2/README.md`

### week3-1

- 주제: 모던 JavaScript 개발 환경, Node/npm/package.json, 실행 루틴 실습
- 시작 파일: `week3-1/README.md`

### week3-2

- 주제: Vue 기초, 상태 기반 UI, 컴포넌트 실습
- 시작 파일: `week3-2/README.md`

## 사용 방식

1. 진행할 회차 폴더 하나를 VS Code에서 바로 연다.
2. 해당 회차의 `README.md`를 먼저 읽는다.
3. `docs` 문서를 순서대로 따라가며 설명한다.
4. 실습 회차는 `src` 폴더에서 코드와 함께 진행한다.

## 비고

- 1주차 회차는 문서 중심 진행이라 `src`에는 안내 문서만 둔다.
- 2주차와 3주차는 실제 예제 소스가 함께 들어 있다.
